âš ï¸ TROUBLESHOOTING

Problem: npm install gagal
Solution: 
- Update npm: npm install -g npm
- Delete package-lock.json
- Coba lagi npm install

Problem: Admin panel blank page
Solution:
- Refresh browser (Ctrl+F5)
- Check console (F12) untuk error
- Restart server

Problem: Upload foto tidak bekerja
Solution:
- Pastikan public/images folder ada
- Pastikan permissions folder OK
- Restart server

Problem: Database error
Solution:
- Delete file data.db
- Jalankan lagi: node admin-setup.js

Problem: Server tidak bisa diakses
Solution:
- Port 3000 mungkin terpakai
- Cek: netstat -ano | findstr :3000
- Kill process atau ubah port di server.js
